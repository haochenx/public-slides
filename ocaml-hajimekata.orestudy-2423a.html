# 01 · OCaml とは ;;

実用的で、汎用の
関数型言語。

  • 1996年、フランスの Inria（国立情報学自動制御研究所 — 欧州最大級の Computer Science 研究機関）で開発。ML / Caml の系譜を継ぐ。
  • 汎用・実用志向のマルチパラダイム。関数型を中心に、命令型も OOP も書ける。
  • Hindley–Milner 型推論による、強くて静的な型システム。型注釈はほぼ書かないのに、型安全。
  • アカデミアにも産業にも確かな実績。F# / Scala / Rust / Gleam などに影響。初期の Rust コンパイラは OCaml 製。
01
# 01 · OCaml とは / 型推論

書かなくても、型は付く。

utop
# let rec fib n =
    if n < 2 then n else fib (n - 1) + fib (n - 2) ;;
val fib : int -> int = <fun>

型注釈はゼロ。それでもコンパイラが int -> int と推論し、間違った使い方はコンパイル時に弾いてくれる。

02
# 02 · どこでも動く ;;

ほぼどこでも動く。

native

ネイティブ

x86-64 / ARM / RISC-V / POWER… 多数の CPU・OS に対応。一部マニアックな環境(S390 + IBM Zとか)まで。

bytecode

バイトコード

移植性の高いバイトコード。
C が動く環境ならだいたい動く。

web / js · wasm

ブラウザ

js_of_ocaml・wasm_of_ocaml で
JS / WebAssembly ランタイムへ。

ポイント：js_of_ocaml は「実際の OCaml バイトコード」を変換するので、ネイティブと意味論がほぼ一致。
F#(Fable) や Scala.js のような「Web 版だけ挙動が違う」問題がない。

# 03 · 採用事例 ;;

実世界で、ちゃんと使われている。

社内の標準言語。最大級の産業ユーザー。
Meta / Facebook

Bloomberg
金融・言語ツールで採用（OCaml 方言である ReScript の源流でもある）。
Docker
一部コンポーネント（MirageOS 由来）。
Tezos
ブロックチェーンの中核実装。
M3（エムスリー）
Web・ML・CLI など複数プロジェクトで採用。日本の医療 IT 大手。
